# CodeAlpha_HangmanGame# 🎮 Hangman Game (Python)

This is a simple text-based Hangman game built using Python.  
The player has to guess a randomly selected word one letter at a time within a limited number of attempts.

---

## 🚀 Features
- Random word selection from a predefined list  
- User guesses one letter at a time  
- Maximum of 6 incorrect attempts  
- Displays correct guesses and blanks for remaining letters  
- Simple and interactive console-based gameplay  

---

## 🛠️ Technologies Used
- Python  
- Concepts: loops, if-else, lists, strings, random module  

---
